ประเภทเครื่องแม่ข่าย (Server Types)

ตัวอย่างและคำอธิบายเครื่องแม่ข่าย 3 ประเภทที่สำคัญ

1️⃣ Web Server

หน้าที่: เป็นตัวกลางให้บริการเว็บไซต์แก่ผู้ใช้ รับคำขอ (Request) จากเว็บเบราว์เซอร์ แล้วส่งข้อมูลหน้าเว็บ (HTML, CSS, รูปภาพ) กลับไปแสดงผล

Web Server ทำให้เว็บไซต์ออนไลน์ได้ตลอดเวลา รองรับผู้ใช้จำนวนมากพร้อมกัน

ตัวอย่างซอฟต์แวร์: Apache HTTP Server, Nginx

การใช้งานจริง: เว็บไซต์บริษัท ร้านค้าออนไลน์ บล็อกส่วนตัว

2️⃣ File Server

หน้าที่: จัดเก็บและแชร์ไฟล์ในเครือข่ายองค์กรหรือในบ้าน ทำให้ผู้ใช้หลายคนเข้าถึงไฟล์เดียวกันได้สะดวก ปลอดภัย

ลดไฟล์ซ้ำซ้อน กำหนดสิทธิ์การเข้าถึงให้เฉพาะผู้ใช้ที่ได้รับอนุญาต

ตัวอย่างซอฟต์แวร์: Samba (แชร์ไฟล์ระหว่าง Windows/Linux), FTP Server (vsftpd, FileZilla Server)

การใช้งานจริง: เก็บเอกสารบริษัท แชร์ไฟล์งานให้ทีม สำรองข้อมูลสำคัญ

3️⃣ Database-Server

หน้าที่: เก็บ จัดการ และให้บริการฐานข้อมูล รองรับการค้นหา แก้ไข เพิ่ม ลบข้อมูลขนาดใหญ่เป็นระบบ

เป็นหัวใจของแอปพลิเคชันต่างๆ เช่น เว็บไซต์ร้านค้าออนไลน์ ระบบลงทะเบียน ระบบบัญชี

ตัวอย่างซอฟต์แวร์: MySQL, PostgreSQL, Microsoft SQL Server

การใช้งานจริง: เว็บไซต์สมาชิก แอปมือถือ ระบบจองตั๋วออนไลน์
